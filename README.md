# Data Gouv

Projet Data Gouv Back :

-   Python 3.7
-   Django
-   Postgres

# First installation

Create virtual env :

> virtualenv --python python3.7 env

# Activate the virtual env (Windows):

> .\env\Scripts\activate

# Install requirements in env:

> pip install -r requirements.txt

## Run DB migrations

> python manage.py runserver

## Run dev server

> python manage.py runserver
